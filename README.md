# ✈️ SkyMetrics — US Aviation Performance Dashboard

## 📌 Project Overview
An interactive Power BI dashboard analyzing 2015 US flight 
performance data across 6M+ flights, 14 airlines, and 300+ airports.
Built as part of Microsoft Elevate Power BI Internship — February 2026 Batch.

## 🗂️ Dashboard Pages
| Page | Description |
|------|-------------|
| 🏠 Executive Overview | KPIs, flight status donut, top airlines |
| ✈️ Airline Performance | Delay analysis and comparison by carrier |
| 🌍 Geographic Intelligence | US airport traffic bubble map |
| 📅 Delay Trend Analysis | Monthly patterns and delay cause breakdown |

## 📊 Key Insights
- 36.43% of all US flights were delayed in 2015
- Delta Airlines had the best on-time rate at 71.24%
- Atlanta (ATL) was the busiest airport in the entire US
- Airline operational issues caused 49% of all delays
- Summer months (Jun-Aug) had the highest flight volumes
- Weather delays spike significantly in winter months
